# 🏥 ViverBem

Bem-vindos à organização do projeto **ViverBem**! Este é um projeto acadêmico desenvolvido por estudantes de Ciência da Computação da Universidade Maurício de Nassau (Recife/PE).

## 🚀 Sobre o Projeto
O ViverBem  é um aplicativo mobile multiplataforma (Android e iOS) que utiliza a câmera do smartphone para identificar medicamentos através de Inteligência Artificial. 
O aplicativo funciona de forma integrada com um backend em nuvem que gerencia os dados do usuário, processa as imagens e armazena o histórico de medicamentos e doses.

Funcionalidades Principais:

* Escaneamento e identificação de medicamentos por IA via câmera
* Exibição de informações detalhadas do medicamento (bula simplificada)
* Cadastro e gerenciamento de alarmes de medicação
* Notificações push com confirmação de dose
* Histórico completo de escaneamentos e doses
* Acessibilidade: alto contraste, fontes ampliadas, leitura por voz
* Cadastro de contato de emergência

O público-alvo primário são pessoas idosas (60 anos ou mais) que fazem uso regular de medicamentos e necessitam de auxílio para identificação e gerenciamento de horários. 

O público secundário inclui cuidadores e familiares que acompanham o tratamento medicamentoso de idosos.

## 🛠️ Stack Tecnológica
Para garantir performance e tipagem segura, estamos utilizando:
* **Frontend:** React + TypeScript
* **Backend:** Node.Js + Python
* **IA:** Google Vision AI
* **Estilização:** Tailwind CSS
* **Roteamento:** React Router
* **Design:** Figma
